HOW TO RUN ON YOUR PHONE (4 steps)
===========================================================

STEP 1: Install Android Studio (if not already)
-----------------------------------------------------------
  - Download: https://developer.android.com/studio
  - Install, open it, go through the setup wizard

STEP 2: Open the TrueNorth project
-----------------------------------------------------------
  - Open Android Studio
  - Click "Open" and navigate to:
      C:\Users\George\Desktop\truenorth_app
  - Click OK
  - Wait for Gradle to sync (bottom bar shows progress)
      *First time may take 5-10 minutes downloading things*

STEP 3: Connect your Android phone
-----------------------------------------------------------
  - On your phone: Settings → About Phone → Tap "Build Number" 7x
  - Go back → Settings → System → Developer Options → enable USB Debugging
  - Plug phone into computer via USB cable
  - On phone: when prompted "Allow USB debugging?" → tap OK
  - In Android Studio: you should see your phone name in the
    top toolbar dropdown (next to the green play button)

STEP 4: Run the app
-----------------------------------------------------------
  - Click the green ▶ Run button (or press Shift+F10)
  - Wait for the build (1-3 minutes)
  - The app installs automatically on your phone!


IF ANDROID STUDIO HAS BUILD ERRORS
===========================================================
The most common fix:

  In Android Studio: Build → Clean Project → Build → Rebuild Project

  If you still see errors, try:
    File → Invalidate Caches → Invalidate and Restart
